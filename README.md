# PhotoGallery

- An app that displays photos from Flickr's copyright-free photo gallery.

### Components
 
- RecyclerView
- Retrofit
- Scalars Converter
- Repository Pattern
- JSON
- GSON
- REST Api from https://api.flickr.com/
- HandlerThread
- WorkManager
- Broadcast Intents
- WebView
- ProgessBar
